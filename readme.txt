CardPerfil - DevMedia Mobile
Um projeto mobile desenvolvido como parte do curso DevMedia, criando um componente de perfil estilizado com informações do usuário.

📱 Preview
<div align="center"> <img src="https://github.com/seu-usuario/cardperfil/blob/main/preview.png?raw=true" alt="Preview do CardPerfil" width="300" /> </div>
✨ Funcionalidades
Perfil personalizável (nome, cargo, foto e descrição)

Layout responsivo para diferentes tamanhos de tela

Botões de interação (seguir, mensagem)

Estilização moderna com shadows e bordas arredondadas

🛠️ Tecnologias
React Native (ou Flutter/Kotlin/Swift - ajuste conforme seu projeto)

Styled Components (ou alternativa de estilização)

Expo (se aplicável)
